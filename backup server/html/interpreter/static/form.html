Reglas y programas de ejemplo
Reglas:
  • cada expresión ocupa una línea, y dicha línea tiene que acabar con un '\n'
  • los if no llevan else explícito, y están compuestos por tres listas (condición) (afirmativa) (negativa)
  • los case y los cond sí pueden llevar else
Programa 1: máximo común divisor
(define gcd (lambda (a b) (if (= a b) a (if (> a b) (gcd (- a b) b) (gcd a (- b a))))))
(gcd 18 90)
Programa 2: expresión case
(case (+ 5 5) ((4 9 1) 'd64) ((1 2) 'pepito) ((10) 'jorgito))
Programa 3: factorial
(define (factorial x) (if (= x 1) 1 (* x (factorial (- x 1)))))
(factorial 900)
